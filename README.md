# quxuedao
启蒙岛 —— 儿童趣味学习社区
项目简介
启蒙岛是一款面向儿童的沉浸式趣味学习网页项目，以童话场景 + AI 互动为核心，打造集文化科普、海洋探索、益智游戏于一体的线上启蒙平台，让孩子在趣味体验中学习知识、感受传统文化与自然奥秘。
功能模块
主地图：场景化入口，一键切换三大主题空间
中华文化文化馆：二十四节气、名胜古迹、民俗艺术可视化展示，AI 向导智能答疑
探索号潜水艇：32 种海洋生物图鉴，海底漫游 + AI 科普互动
梦幻森林乐园：AI 故事生成、英语单词学习、钢琴模拟、萌宠对话
技术特色
纯前端实现，HTML+CSS+JS 构建交互界面，搭配本地 Ollama 大模型提供 AI 对话、内容生成能力，界面卡通治愈、操作简单易懂，适配全年龄段儿童使用。
